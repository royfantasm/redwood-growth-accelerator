# Redwood Growth Accelerator

A complete digital marketing service website with comprehensive tracking and analytics.

## Live Site
The website is deployed via GitHub Pages.

## Features
- 4 complete pages (Home, Services, Resources, Contact)
- Responsive design for all devices
- Google Analytics 4 + Tag Manager integration
- Facebook Pixel tracking
- Custom funnel tracking
- Dark mode support
- Accessibility compliant

## Pages
1. **Home** - Hero section, services preview, testimonials
2. **Services** - 4-tier service comparison, FAQ, timeline
3. **Resources** - Free templates, tools, newsletter signup
4. **Contact** - Contact form, Calendly integration, next steps

## Deployment
This site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
